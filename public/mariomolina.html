¿Quiénes somos?

Fundada en 2015, Generation es una organización sin fines de lucro que tiene como objetivo combatir el desempleo juvenil a través del desarrollo de programas de capacitación para el empleo. Con operaciones en más de 20 ciudades de 8 países, hemos graduado a más de 20,000 jóvenes en todo el mundo.

A través de nuestra probada metodología de enseñanza, ayudamos a nuestros alumnos a desarrollar las competencias, mentalidades y comportamientos necesarios para tener una carrera profesional exitosa.

El programa de desarrollo fullstack en Java te ayuda a desarrollar lógica de programación, habiliades de desarrollo de plataformas en Java y un nivel de inglés suficiente para trabajar en la industria. Adicionalmente, al terminar el curso, se te garantiza una entresvista laboral con empleadores líderes de la industria, como:
  • Oracle
  • Rappi
  • Wizeline
  • Intel

    Contenido del curso

    El curso se compone de sesiones de:
  • Teória y práctica de programación que te ayudan a desarrollar las hablididades técnicas necesarias
  • Ingles enfocado en el mundo de la tecnología
  • Habilidedes y comportamientos necesarios para desenvolverte de manera exitosa en un trabajo

    • Adicional al contendio de las sesiones, todos nuestros alumnos cuentan con el apoyo de una mentora que los guiará durante todo el proceso y los ayudará a desarrollar las mentalidades y comportamientos necesarios para alcanzar su máximo potencial. El equipo de mentoría también ayuda a nuestros alumnos a construir un perfil profesional (CV, GitHub, LinkedIn, etc.) así como a realizar prácticas y simulaciones de entrevista para familiarizarse con el proceso y mazimizar las probabilidades de una contratación.


    El contenido técnico abarca diversos temas; pero los puntos principales que aprenderás dentro del programa son:

    Módulo 1. Preparación previa

    • Arquitectura de las aplicaciónes web y tecnologías disponibles
    • Gestion del software y de los proyectos de desarrollo
    • Algoritmos
    • Fundamentos del desarrollo orientado a objetos
    • Fundamentos de la programación Java

    Módulo 2. Implementación de aplicaciones web

    • Fundamentos UX y tecnologias Front-End (Bootstrap, AngularJS, ReactJS)
    • Servudores Google y Amazon
    • Aplicaciones web y MVC
    • Testing de aplicaciones y aseguramiento de calidad
    • Despliegue de aplicaciones
    • Diseño de bases de datos relacionales y no relacionales
    • Servidores de bases de datos e implementación de la capa de Persistencia
    • Aplicaciones AJAX

    Módulo 3. Desarrollo avanzado

    • Patrones de diseño y capa de Business Logic
    • Aplicaciones basadas en JSF y Struts
    • Aplicaciones basadas en Spring Framework / SpringBoot
    • Persistencia basada en JPA e Hibernate

    Módulo 4. Servicios web

    • SOA y Servicios Web
    • Servicios Web SOAP
    • Servicios Web RESTFul
    • Clientes SPA
